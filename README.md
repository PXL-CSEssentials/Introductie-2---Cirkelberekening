# Introductie 2: Cirkelberekening

Deze oefening dient om na ingave van de straal de oppervlakte en de
omtrek van een cirkel te berekenen.

Er is een tekstvak *radiusTextBox* om de straal in cm in te geven.

Er zijn de tekstvakken *surfaceTextBox* en *circumferenceTextBox* om de resultaten
weer te geven.

Er is een opdrachtknop *calculateButton* en een opdrachtknop *clearButton*.
